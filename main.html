Time, Space and Complexity in Brief

Time complexity is function that tells about how runtime required to completely execute a program increases as input size of program in increased its not the time required to execute the program. It is function that maps program's input size to runtime required for execution of program.It helps identify how efficient our program is calculating the results for greater number of input size

Space Complexity is function that tells about how memory is required to complete program execution increases as input size of program is increased. Space complexity does not consider OS/ compiler overhead, cpu cache created while executing program. For ex: linear search algorithm as space complexity of O(n)
